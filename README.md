# CRM-JURIDICO

## Project Overview

This is a CRM system tailored for legal professionals. It aims to provide tools for managing client relationships, case files, appointments, and invoicing. The project is structured to facilitate both backend and frontend development, along with a dedicated database setup.

## Folder Structure

- **backend/**: Contains the server-side code and APIs.
- **frontend/**: Contains the client-side code and user interface.
- **database/**: Configuration and scripts related to the database setup.

## Getting Started

To get started with this project, clone the repository and follow the instructions in the respective folders.
